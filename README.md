PipelinePro is a full-stack game production dashboard built for Associate Producers and Production Managers to track every phase of a game's development lifecycle. Here's what it can do:

Game Pipeline
Track your project across all development phases — Concept & Pitch, Pre-Production, Production, Alpha/Beta, Launch, and Post-Launch/LiveOps. Each phase has its own detail page where you can view tasks by status, add post-mortem entries, and see resolved risks tied to that phase.

Blocker Heatmap
Log and manage blockers across departments and phases. Assign severity levels (low/medium/high/critical), link to a responsible owner, and mark them resolved. Resolving a blocker automatically writes a "Blocker Cleared" event to the timeline.

Milestone Buffer Tracker
Track scheduled vs. actual milestone dates with a calculated buffer. Each milestone shows whether it's On Track, At Risk, or Slipping based on the remaining buffer days. Recovering a milestone to on-track auto-logs a timeline event.

Department Pulse
Monitor team health across all departments with a weekly morale and velocity score. Entries show trends over time so you can spot burnout or productivity drops early.

Risk Register
Create and manage risks with probability, impact, phase linkage, and mitigation notes. Active risks are shown by default. When a risk is mitigated or closed, it moves to a collapsible Mitigated Archive with its resolution notes and timestamp. Resolving a risk logs an event to the timeline.

LiveOps Health Bar
Log post-launch live service metrics — DAU, crash rate, server uptime, and revenue — to monitor the health of the game after it ships.

Post Mortem & Retrospective
Each phase detail page has a retrospective section for logging Lessons Learned, Notes, and Action Items. Resolved risks from that phase automatically appear here for reference during reviews.

Project Timeline
A chronological log of every significant event across the project — blockers cleared, risks resolved, milestones recovered, and lessons learned. Events are created automatically; you can also add manual milestone or launch markers. Green "Cleared" flags mark major hurdles overcome.

Secure Login
The entire app is protected by login via Replit Auth. Access requires signing in with Google, GitHub, Apple, or email. Your session persists for a week and your profile is shown in the sidebar with a logout button.
